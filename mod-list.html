Debug & Runtime Tools

Run before launching the game to unlock features not available by default

These tools run alongside the game rather than modifying your Downloads folder. They patch the game in memory at runtime. Run before launching, keep open while playing, close when done.

Sims 1 Legacy Collection Hacks

by FaithBeam — Unlocks the hidden edit_char debug cheat. Run Sims1LegacyHacks.exe before launching the game and keep it open while you play. Works on Steam and EA App.

✅ Unlocks: Change Sim appearance (face, hair, skintone) in live mode via full Create-A-Sim access for any Sim.

Download & run

Download from GitHub, extract, run Sims1LegacyHacks.exe before launching the game. Keep it open while you play.

Open the cheat console

In-game press Ctrl+Shift+C.

Type edit_char

Type edit_char and press Enter, then click any Sim to open Create-A-Sim for them.

View on GitHub ↗

Children Ageing Up (with Birthday Cake mod)

Combined with the FaithBeam tool and a birthday cake mod, children can be aged up to adults. Using edit_char afterwards gives full control over the adult's appearance.
