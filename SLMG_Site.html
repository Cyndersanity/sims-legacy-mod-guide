Installing Mods

Folder paths, rules, and save file location

💡
Unlike Sims 4, mods go inside the game's install directory — not your Documents folder. The subfolder names are the same for both EA App and Steam installs.
🎮 Steam Install
Right-click the game → Manage → Browse local files to open this folder.
C:\Program Files (x86)\Steam\steamapps\common\The Sims Legacy Collection\
🟧 EA App Install
In EA App, right-click the game → Manage → Open install folder.
C:\Program Files\EA Games\The Sims Legacy\

What Goes Where

File TypeDestination SubfolderNotes
.iff & .FAR filesDownloads\The vast majority of mods go here
.BMP .CMX .SKN (skins)GameData\Skins\⚠ NO subfolders — files must sit directly in this folder
.WLL (wall coverings)GameData\Walls\Custom wallpaper & walls
.FLR (floor tiles)GameData\Floors\Custom floor textures
Game file overridesGameData\Objects\For mods that replace existing game objects
Expansion overridesExpansionPack#\# = expansion number (1–7)
🔄
Using old mod instructions? Replace C:\Program Files (x86)\Maxis\The Sims\ with the Steam or EA App path above. All subfolder names are identical.
💾
Save files live here — back them up before installing any mods!
C:\Users\%USERPROFILE%\Saved Games\Electronic Arts\The Sims 25\
⚠️
Critical rules: Skin files must be placed directly in GameData\Skins\ with no subfolders. Always unzip files before placing them. Add mods one at a time so you can easily spot any that cause problems.

Creating Custom Skins, Clothes & Hair

A beginner-friendly guide to making your own Sims 1 content

📌
Sims 1 is different from later games. A Sim's body and clothing are a single texture file. Hair is painted directly onto the head skin. You can't mix and match tops and bottoms — each outfit is a complete skin covering the whole body.
🛠️ The Sims Creator (TSC)
The official Maxis CC creation tool. Provides a 3D rotating Sim body to paint onto and automatically exports the correct game-ready files. This is your starting point for all clothing and hair creation.
Find it by searching "Sims Creator" on ModTheSims or The Sims Resource.
🖼️ GIMP — Free Image Editor
For painting and editing textures in detail outside of TSC. Critical: before exporting your file, you must convert it: Image → Mode → Indexed (256 colours). Regular BMPs show as a blank white body in-game.
↗ gimp.org — free download

Making Custom Clothing — Step by Step

Open The Sims Creator and choose a body mesh

Pick the body type you're designing for: fit/fat/skinny, male/female, adult/child. Each needs a separate outfit — a fit female outfit won't work on a fat female body without a separate version.

Export the base texture to your image editor

You'll see the "UV layout" — a flat map of how the 2D texture wraps around the 3D body. The front torso is in the centre; arms are in strips to the sides; legs at the bottom.

Paint your outfit on top in GIMP or Photoshop

Work in layers — keep the base skin on one layer and paint your clothing on top. Use the Warp transform tool to fit patterns and textures around the body shape. Pay attention to edges where body parts join — seam misalignments show clearly in-game.

Make all three skin tone versions

Sims 1 has light, medium and dark skin tones. If your outfit covers the full body, make three copies — one per skin tone. Swap the base skin layer underneath your clothing for each and re-export.

Save as Indexed BMP — this is critical!

In GIMP: Image → Mode → Indexed → Export as .BMP. In Photoshop: Image → Mode → Indexed Colour → Save As .BMP. Do not save as a regular 24-bit BMP, PNG or JPG — the game will show it as a blank white body.

Import back into TSC and export the final files

TSC generates the .BMP, .CMX and .SKN files. Place all three directly inside GameData\Skins\ with no subfolders whatsoever.

IFF Hacking — Creating Gameplay Mods

Build your own objects, hacks and interactions

Every object in Sims 1 is an .iff file containing visual sprites, behaviour scripts (BHAV), numeric values (BCON) and text labels (STR#). IFF hacking means editing these to change how objects work — or creating new ones entirely.

⚡
Always clone an object before editing it. Use Transmogrifier to create a copy of any game object as a new .iff file. Never modify original game files directly — you may not be able to recover them.

What's Possible — Engine Limits

Honest answers about what can and can't be done

✅
Change outfit anytime via a mirror (intermediate IFF hack)
Outfit changes through a Dresser or Wardrobe are already built into the base game. Adding this to a mirror requires cloning the mirror .iff and pointing it to the dresser's existing outfit-change behaviour routine in IFF Pencil 2. No standalone mod exists yet — check Simblr.cc, or try it yourself following the IFF Hacking section above.
⚠️
Change Sim appearance (face/hair/skintone) in live mode
Not possible in live mode via any current mod. Workaround: Use The Sims Creator to edit and replace a Sim's head or body skin files outside the game, then reload your save. A true in-game "makeover mirror" would need a behaviour that swaps skin file references — advanced territory that hasn't been cracked yet. Watch Simblr.cc for developments.
❌
Children growing into Teens, Teens into Adults (Sims 2-style aging)
The Sims 1 engine has only three life stages: Baby, Child, and Adult. The teen life stage didn't exist until Sims 2. Adding it would require modifying the core game executable (sims.exe), which is encrypted in the Legacy Collection. The only built-in child ageing path is the Makin' Magic "Age of Instant" charm — which ages a child to adult but fully randomises their appearance. For full generational aging, The Sims 2 Legacy Collection (included in the same bundle) has the complete baby → toddler → child → teen → adult → elder system.
❌
Edge scrolling on dual monitors
Edge scrolling requires Fullscreen mode. Dual monitors require Windowed or Windowed Fullscreen mode. These are mutually exclusive at engine level — no mod can fix this. Workaround: use WASD or the arrow keys for camera panning. Go to Game Options → Graphics and set "Windowed Fullscreen" for the best dual-monitor experience.
✅
Higher resolution and sharper-looking sprites
The March 2025 patch already added fractional scaling — at 4K, the game renders at 3× native scale for noticeably sharper sprites. Make sure your game is fully updated and set to your monitor's native resolution in Game Options → Graphics. Community texture upscale packs are beginning to appear on Simblr.cc and Nexus Mods.

Troubleshooting

Common problems and how to fix them

📁 Mods not showing in game
  • Unzip everything first — the game cannot read .zip files directly
  • Skin files must sit directly in GameData\Skins\ — no subfolders at all
  • Check the file extension is correct (.iff, not .iff.zip)
  • Some mods need GameData\Objects\ or an ExpansionPack\ folder — always read the readme
💥 Game crashes after a new mod
  • Remove all new mods, then add them back one at a time
  • Check Simblr.cc and EA Forums for patch 1.122 compatibility notes
  • Look for a Legacy-compatible update on the creator's page
  • Try adding to a fresh save rather than your main neighbourhood
🤍 Skins show as a white blank body
  • Your BMP is not saved as Indexed (256-colour) format
  • GIMP: Image → Mode → Indexed → Export As BMP
  • Photoshop: Image → Mode → Indexed Colour → Save As BMP
  • Do not use PNG, JPG or 24-bit BMP formats
📦 Old mod won't work in Legacy
  • Use the path translation in the Install section above
  • Mods that patch sims.exe directly won't work — the EXE is encrypted in Legacy
  • Check the mod creator's page for a Legacy-specific update
  • Post in the EA Forums Sims 1 section for community help
