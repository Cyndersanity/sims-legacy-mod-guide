How Walls, Floors & Roofs Work

The easiest type of CC to make

Wall and floor CC doesn't require any mesh files, UV layouts, or 256-colour restrictions. You create an image in any paint program, run it through Home Crafter or save it directly, and the game reads it. It's genuinely the most accessible entry point into Sims 1 CC creation.

🧱 Walls – .WLL files
Wall coverings use the .WLL extension. They go in GameData\Walls\ and subfolders are supported – you can organise by style or creator. Home Crafter generates .WLL files from any image you import.
🟫 Floors – .FLR files
Floor tiles use the .FLR extension and go in GameData\Floors\. Subfolders are supported here too. Home Crafter generates .FLR files the same way as walls.
🏠 Roof textures – .BMP files
Roof textures are small 32×32 .BMP files that go directly in GameData\Roofs\. No subfolders – the game won't read them otherwise. These are made manually in GIMP without needing Home Crafter.
📐 Image dimensions
Wall textures are 128×232 pixels – they are not square. Floors tile correctly as square images at 128×128. Roof textures must be exactly 32×32px. HomeCrafter handles scaling and conversion automatically for walls and floors.
🛠️

Getting Home Crafter Working

Needs the registry fix and compatibility settings

Home Crafter needs the Legacy Collection registry fix plus two compatibility settings on its .exe: 16-bit colour mode and Windows XP SP2 compatibility. The Feraligatr installer handles all of this automatically.

⚠️ Home Crafter specifically requires both compatibility settings. The "16-bit colour" error is the most common issue. If you see it after running the installer, right-click HomeCrafter.exe → Properties → Compatibility tab → tick 16-bit colour and set mode to Windows XP (Service Pack 2).

Route A – Feraligatr Installer Recommended

Download and run the installer

Get it from github.com/Secondhand-Feraligatr/Sims-1-Programs-for-Legacy-Collection. Choose Custom Install → Select manually and include Home Crafter. Tools install to a Programs folder inside your Legacy Collection directory.

If you get a 16-bit colour error

Right-click HomeCrafter.exe → Properties → Compatibility tab → enable Run in 16-bit colour mode and set compatibility mode to Windows XP (Service Pack 2). Click OK.

Route B – Manual

Download Home Crafter from the Internet Archive, apply the registry fix, and set the two compatibility settings manually as described above.

ℹ️ For roof textures only – no Home Crafter needed. Roof textures are plain 32×32 Indexed BMP files made directly in GIMP. No tool setup required.
💡 Browser-based alternative: If HomeCrafter gives you trouble, SneakySims Wallpaper Maker and SneakySims Floor Maker run entirely in your browser – no download, no registry fix, no compatibility settings. They produce .WLL and .FLR files tested with the Legacy Collection.
🧱

Making a Wall Covering

Create your design in GIMP

For walls, create a 128×232 pixel image (not square – walls are taller than they are wide). For floors, use 128×128 pixels. Save as PNG or any standard format – HomeCrafter handles the conversion to the game’s format.

Open Home Crafter

Launch it from the Programs folder in your Legacy Collection install (if using the Feraligatr installer) or from your Start menu. Select Wall Coverings from the main menu.

Import your image

Click Import and select your image file. Home Crafter shows a preview of how it tiles on a wall in game. You can see both the interior and exterior views.

Fill in the details and export

Give your wall covering a name and description (these appear in the game's Build mode catalogue). Click Export – Home Crafter generates the .WLL file.

Place in GameData\Walls\

Copy the .WLL file into GameData\Walls\ in your Legacy Collection install folder. Subfolders are supported – you can organise by creator or style.

🟫

Making a Floor Tile

The process is identical to walls but select Floor Tiles in Home Crafter instead of Wall Coverings. A few tips specific to floors:

ℹ️ Floor tiles repeat across the room – your design will tile. Images that repeat correctly (solid colours, geometric patterns, regular textures) look better than images with a clear centre or focal point.

Design with tiling in mind

Open your image in GIMP and use Filters → Map → Tile to preview how it repeats. Adjust the edges so colours blend at the seams if needed.

Import into Home Crafter and export as .FLR

Same process as walls – open Home Crafter, select Floor Tiles, import your image, name it, export. The output is a .FLR file.

Place in GameData\Floors\

Copy the .FLR file to GameData\Floors\. Subfolders supported here too.

🏠

Making a Roof Texture

No Home Crafter needed – GIMP only

Roof textures are tiny 32×32 pixel BMP tiles. The game tiles them across the roof surface. You make them directly in GIMP – no Home Crafter, no registry fix.

⚠️ Roof textures must be saved as Indexed (256-colour) BMP, exactly 32×32 pixels, and placed directly in GameData\Roofs\ with no subfolders. Any of these wrong and the texture won't appear.

Create a 32×32 image in GIMP

File → New → set Width and Height both to 32 pixels. Design your roof tile – roof tiles are small so keep the design simple: a brick pattern, shingles, thatch, concrete etc. Consider that it will tile correctly across a large surface.

Convert to Indexed 256-colour

Image → Mode → Indexed → Maximum colours: 255 → Convert.

Export as BMP

File → Export As → name your file (e.g. DarkSlate.bmp) → select BMP format → Export. Leave BMP options at their defaults.

Place in GameData\Roofs\

Copy the .BMP directly into GameData\Roofs\. No subfolders. The texture will appear in the Build mode roof picker the next time you load the game.

⚠️

Common Mistakes

ProblemCauseFix
Wall / floor not appearingFile in a subfolder inside Walls or Floors (not valid – subfolders are supported but the game must see the files directly in Walls/ or Floors/)Move the .WLL / .FLR to the correct root folder, or to a single subfolder level inside it
Home Crafter won't open16-bit colour mode or XP SP2 compatibility not set, or registry entry missingUse the Feraligatr installer, or set both compatibility settings manually on HomeCrafter.exe
Roof texture not appearingFile in a subfolder, wrong size, or saved as 24-bit BMPMust be 32×32px, Indexed 256-colour BMP, directly in GameData\Roofs\ root
Roof texture looks blurry/wrongFile saved at larger size and scaled down by the gameDesign natively at 32×32 – the game doesn't scale roof textures up
