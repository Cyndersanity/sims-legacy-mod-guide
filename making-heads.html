How Hair & Heads Work

⚠️ Sims 1 has no separate hair item. Hair is painted directly onto the head texture – to add a new hairstyle, you paint it onto the head file. To add a new hair colour, you make a separate file with the colour repainted.
💇 One file = face + hair + neck
The head texture contains the Sim's entire head appearance – face, ears, neck, and hair all in one flat bitmap. The hair is part of the image, not a mesh on top of it.
🎨 Three skin tone versions
Head files need a light, medium and dark version, just like body skins. The skin tone affects the face and neck areas – the hair colour portion stays the same across all three.
👤 The C prefix
Head files use the C prefix (Cranium). The filename format mirrors body skins: Cfa skn lgt_HairName.bmp = female adult, skinny body type, light skin. They go in the same GameData\Skins\ folder as body skins.
📸 Face Photo Wizard
The Sims Creator includes a Face Photo Wizard that maps a real photo onto the head mesh template, letting you create a custom face texture. Good for character-specific heads.
🛠️

Getting the Tools Working

Same setup as clothing – registry fix required

The same registry fix described on the Making Clothing – Getting Tools Working section applies here. The Sims Creator is the main tool for head CC; GIMP handles the texture painting.

ℹ️ If you already set up the Feraligatr installer for clothing, everything you need is already installed. No additional setup needed for heads.
📦

What You Need

The Sims Creator For CMX/SKN files
Select a head mesh in TSC and export the head template BMP to paint over. Import your finished texture back in and TSC exports the .CMX and .SKN companion files.
GIMP or Photoshop Image editing
For painting the hairstyle and face. Head skin BMPs are 128×128 pixels, Indexed 256-colour. In GIMP: Image → Mode → Indexed (255 colours) → Export as .BMP.
↗ gimp.org
Existing head files As a base
Extract a head BMP from GameData\Skins\ (any file starting with C) to use as a painting base. This ensures correct proportions and UV layout from the start.
🚶

Step-by-Step: Making a Head / Hair Skin

Get a head template

Either export a head template from The Sims Creator, or copy an existing C-prefix file from GameData\Skins\ to use as a starting point. Open it in GIMP.

Paint your hairstyle and face

Work in layers. The head template shows you where the face, ears, scalp, and neck sit. Paint the hair over the scalp and around the face as appropriate for your style. Face details (eyes, nose, mouth) are also part of this same image.

Make the medium and dark skin tone versions

The hair colour area stays the same. Only the face and neck skin colour changes between tones. Adjust the exposed skin areas for medium and dark, keeping the hair layer identical.

Convert to Indexed 256-colour BMP

GIMP: Image → Mode → Indexed (max 255 colours) → Export As .BMP.
Photoshop: Image → Mode → Indexed Colour (256 colours) → Save As BMP.
This step is not optional – 24-bit BMP will show as white in game.

Import into TSC and export

Import your finished head BMP into The Sims Creator. TSC generates the .CMX and .SKN mesh companion files. Export all three.

Name and place in GameData\Skins\

Use the correct C-prefix filename format: Cfa skn lgt_HairName.bmp. Place all three file types directly in GameData\Skins\ – no subfolders. The head appears in the Create-A-Sim face selection.

📸

Face Photo Wizard

Using TSC to map a photo onto the head mesh

The Sims Creator includes a Face Photo Wizard that takes a frontal photo of a face and maps it onto the game's head mesh template. It handles the UV warping automatically.

Open the Face Photo Wizard in TSC

In The Sims Creator, select the Face Photo Wizard option from the main menu.

Import a clear frontal photo

Use a well-lit, straight-on portrait with minimal shadows. The wizard works best when eyes, nose and mouth are clearly visible and centred.

Align the feature markers

TSC will ask you to place markers on the eyes, nose and mouth in the photo. Accurate placement gives better results.

Export and refine in GIMP

The wizard produces a head BMP you can then open in GIMP to clean up the edges, paint in the neck, and add hair.

Custom Head Meshes

Create new hairstyles and head shapes in 3D

The steps above cover repainting existing head textures. If you want to create entirely new hairstyles or head shapes – such as updos, braids, hats, or head accessories – you need to edit the 3D head mesh itself. The workflow is the same as custom clothing meshes with a few key differences.

How head meshes differ from body meshes

C-prefix filenames
Head meshes use the C prefix (e.g. xskin-c001fafit_ponytail-HEAD-HEAD.skn) instead of the B prefix used for body meshes.
HEAD-HEAD bone group
Body meshes are weighted to PELVIS-BODY. Head meshes are weighted to HEAD-HEAD. This means the entire head mesh moves as one unit when the Sim turns or nods – individual facial animation isn't supported.
Neck seam matching
The bottom edge of your head mesh must line up exactly with the top of the body mesh it'll be paired with. Mismatched neck seams cause visible gaps between head and body in game.
Smaller vertex count
Head meshes are typically much simpler than body meshes – the game renders them at low resolution. Keep geometry minimal, especially for complex hairstyles.

Workflow

Use the same tools as clothing meshes – TS1 Blender IO (modern, recommended) or MilkShape 3D (classic). The full step-by-step is on the Making Clothing – Custom Meshes page; just substitute the C-prefix naming and HEAD-HEAD bone group.

ℹ️ Where to find base head meshes: Head SKN files are in GameData\Skins\, often inside FAR archives. Extract with FARx. Look for files starting with xskin-c to find head meshes to use as a starting point.
