Changelog

Most recent changes first

This page tracks all meaningful content changes to the guide. Minor fixes like typos and formatting tweaks aren't listed individually - only content additions, corrections and structural changes. If you spotted something wrong and submitted feedback, check here to see if it's been addressed.
Date Page What Changed
Mar 2026 New pages Added four new pages: Cheats (all working codes, tips and FaithBeam debug unlock), Terminology (what CC, mods, hacking and IFF actually mean - including the Complete Collection vs custom content CC confusion), Updates & Roadmap (this page), and Contact with an embedded Google Form for feedback and suggestions.
Mar 2026 Site-wide Ageing up children & edit_char: Reddit user InsightsIE flagged that children ageing up and live-mode appearance changes are now possible in the Legacy Collection. The underlying tool was created by FaithBeam - their Sims1LegacyCollectionHacks patches the game exe to unlock the hidden edit_char debug cheat. Combined with a birthday cake mod (creator/s still to be confirmed - Puberty Blows by ed95 on MTS is one option), children can now be aged up without Makin' Magic. Updated across What's Possible, Mods, and Cheats pages.

Installing Mods: Expanded to cover all possible file destination folders, added engine CC limits (caps on objects, walls, floors, downloads folder size), and better step-by-step extraction instructions.

IFF Hacking: Added full step-by-step instructions to the Custom Careers section.

Sites: Removed dead links, added Sims File Share (YoNonna), Sims Freaks and Wayback Machine CC resources.

Creating CC: Archive.org download links added for The Sims Creator and Home Crafter. Tool installation steps added.

Roadmap

What's being worked on and what's planned

This is a rough public roadmap. Things in "planned" will happen when time allows. Things in "considering" depend on whether enough people find them useful - if something on this list matters to you, let me know via the contact form.

Currently working on

In progress
Screenshots for tool instructions
Adding screenshots to the IFF Hacking and Creating CC pages showing what the tools look like and where to click. Waiting on captures of Transmogrifier, IFF Pencil 2, Career Creator 3, The Sims Creator and Home Crafter.
In progress
Subpages for tools
Breaking the IFF Hacking page into individual guide pages per tool - one for Transmogrifier, one for IFF Pencil 2, one for Constant Contraption etc. Will include a dropdown nav menu when hovering over the IFF Hacking link.

Planned

Sims 2 Legacy Collection guide

Adding hover dropdown menus to nav items like IFF Hacking and Creating CC so subpages are accessible directly from the header without needing to visit the main page first.

The Puberty Blows mod (ed95 on ModTheSims) and the SimLifeCycle birthday cake mod still need working download links confirmed. The Wayback Machine archive and MTS thread need testing.
Planned
Expansion pack reference page
A quick reference page for what each of the 7 expansions adds - which folders they unlock, which mods need them, and what the EP numbering (EP1-EP7) refers to in mod readmes.
Planned
FAR file guide
A dedicated section on working with FAR files - extracting them with FARx, why some mods bundle files this way, and when you might want to do it yourself.
Planned
BHAV scripting introduction
A beginner introduction to reading and writing BHAV trees in IFF Pencil 2 - the basics of how object interactions are scripted and how to add simple new ones.
