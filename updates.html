ℹ️ This is a rough public roadmap for the guide – not EA game updates. For official patch notes see ea.com/games/the-sims/news. If something on this list matters to you, let me know via the contact form.
🔨

Currently Working On

In progress
Screenshots for tool instructions
Adding screenshots to the IFF Hacking and Creating CC pages showing what each tool looks like and where to click. Waiting on captures of Transmogrifier, IFF Pencil 2, Career Creator 3, The Sims Creator and Home Crafter.
In progress
Subpages for IFF Hacking tools
Breaking the IFF Hacking page into individual guide pages per tool – one for Transmogrifier, one for IFF Pencil 2, one for Constant Contraption etc., each with step-by-step instructions.
In progress
Verify birthday cake mod links
The Puberty Blows mod (ed95 on ModTheSims) and the SimLifeCycle birthday cake mod both need working download links confirmed. The Wayback Machine archive and MTS thread need testing.
📋

Planned

Will happen when time allows

Planned
Sims 2 Legacy Collection guide
A separate companion guide for The Sims 2 Legacy Collection, covering .package files, SimPE, Bodyshop, neighbourhood corruption, and the different folder structure. Same visual design as this guide.
Planned
FAR file guide
A dedicated section on working with FAR files – extracting them with FARx, why some mods bundle files this way, and when you might need to do it yourself.
Planned
BHAV scripting introduction
A beginner introduction to reading and writing BHAV trees in IFF Pencil 2 – the basics of how object interactions are scripted in Sims 1 and how to add simple new ones.
💭

Considering

Depends on interest – use the contact form to vote

Considering
Neighbourhood editing guide
How to edit the neighbourhood map, add terrain, and manage the UserData folder numbering system.
Considering
Lot sharing guide
How to package and share custom lots, and how to import downloaded ones correctly via the UserData/Houses folders.
Considering
Object cloning walkthrough
A worked example from a blank Transmogrifier clone through to a finished modified object with changed values – something to follow along with rather than just read.
Considering
Skin tone matching guide
How skin tone slots work in Sims 1 and how to match custom skins to the correct in-game categories so they appear in the right place in Create-A-Sim.
