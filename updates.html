Changelog

Most recent changes first

Content additions, corrections and structural changes are logged here. Typo fixes and minor tweaks aren't listed. If you flagged something via the contact form, this is where to check whether it's been picked up.

Date Page What Changed
Mar 26, 2026 Site-wide Major content update across the whole guide. Full list of changes:
  • FaithBeam hacks corrected site-wide — it's a runtime memory patcher, not an exe patch. Run it before the game, keep it open while you play. Steamless not required. Works on Steam and EA App.
  • Transmogrifier attribution fixed — created by Don Hopkins (Maxis), not Simlogical. Download link updated to The Sims Depot.
  • SimEchoes removed — site status unconfirmed.
  • Download Sites: SomeSimThings added to active sites; Simblr.cc Tutorials and Dead Site Repository cards added; ModTheSims noted as forum-only (no Sims 1 downloads section); Corylea tag updated to March 2026.

  • About Legacy — Technical Changes: Added native mouse wheel zoom, Alt+Enter resolution cycling, encrypted config files, EA intro video removed (and why the restore workaround doesn't work in Legacy).
  • About Legacy — new In-Game Settings section: F5 options menu reference (Graphics, Sound, Play tabs), full launch flags table, notes on what can't be configured.
  • What's Possible: Mouse wheel zoom added as confirmed improvement; edit_char and child ageing rows promoted from "partial" to confirmed working; 1080p = identical to original now noted in resolution row.
  • Creating CC: Body types corrected (skinny/fit/fat + child — no pregnant mesh); body types card expanded with full explanation.
  • Community help sections: Simscord added to both troubleshooting pages; r/thesims1 confirmed in CC troubleshooting.
  • Mod List: Bunk Beds, Elixir of Eternity (Vampires), Improved Genie Lamp added to Objects section; Obsessed Fan Alarm updated; section renamed to Debug & Runtime Tools.
  • Tools: FaithBeam card fully rewritten; Steamless card updated (no longer required for debug cheats).
Mar 23, 2026 New pages Added four new pages: Cheats (all working codes, tips and FaithBeam debug unlock), Terminology (what CC, mods, hacking and IFF actually mean - including the Complete Collection vs custom content CC confusion), Updates & Roadmap (this page), and Contact with an embedded Google Form for feedback and suggestions.
Mar 22, 2026 Site-wide Ageing up children & edit_char: Reddit user InsightsIE flagged that children ageing up and live-mode appearance changes are now possible in the Legacy Collection. The underlying tool was created by FaithBeam - their Sims1LegacyCollectionHacks patches the game in memory at runtime to unlock the hidden edit_char debug cheat (no permanent exe modification). Combined with a birthday cake mod (creator/s still to be confirmed - Puberty Blows by ed95 on MTS is one option), children can now be aged up without Makin' Magic. Updated across What's Possible, Mods, and Cheats pages.

Creating CC: Archive.org download links added for The Sims Creator and Home Crafter. Tool installation steps added.

Roadmap

What's being worked on and what's planned

This is a rough public roadmap. Things in "planned" will happen when time allows. Things in "considering" depend on whether enough people find them useful - if something on this list matters to you, let me know via the contact form.

Currently working on

In progress
Screenshots for tool instructions
Adding screenshots to the IFF Hacking and Creating CC pages showing what the tools look like and where to click. Waiting on captures of Transmogrifier, IFF Pencil 2, Career Creator 3, The Sims Creator and Home Crafter.
In progress
Subpages for tools
Breaking the IFF Hacking page into individual guide pages per tool - one for Transmogrifier, one for IFF Pencil 2, one for Constant Contraption etc. Will include a dropdown nav menu when hovering over the IFF Hacking link.
In progress
Game crash monitor and log tool
A standalone tool/app that runs alongside the game and monitors for crashes. When a crash occurs it captures a log - what mods were loaded, what was happening in-game, system state - and saves it in a readable format. No tool like this has ever existed for Sims 1. The goal is to help players and mod creators identify exactly what caused a crash rather than guessing. No release date yet, but it's actively being built.

Planned

Planned
Sims 2 Legacy Collection guide
A separate companion guide for The Sims 2 Legacy Collection, covering .package files, SimPE, Bodyshop, neighbourhood corruption, and the different folder structure. Same visual design as this guide.
Planned
Dropdown navigation
Adding hover dropdown menus to nav items like IFF Hacking and Creating CC so subpages are accessible directly from the header without needing to visit the main page first.
Planned
Verify birthday cake mod links
The Puberty Blows mod (ed95 on ModTheSims) and the SimLifeCycle birthday cake mod still need working download links confirmed. The Wayback Machine archive and MTS thread need testing.
Planned
FAR file guide
A dedicated section on working with FAR files - extracting them with FARx, why some mods bundle files this way, and when you might want to do it yourself.
Planned
BHAV scripting introduction
A beginner introduction to reading and writing BHAV trees in IFF Pencil 2 - the basics of how object interactions are scripted and how to add simple new ones.

Considering

Considering
Neighbourhood editing guide
How to edit the neighbourhood map, add terrain, and manage the UserData folder numbering system. Depends on interest - let me know via the contact form if this is something you'd use.

Lot sharing guide
How to package and share custom lots, and how to import downloaded ones correctly using the UserData/Houses folders.
Considering
Object cloning walkthrough
A worked example going from a blank Transmogrifier clone through to a finished modified object with changed values - something to follow along with rather than just read.

How skin tone slots work in Sims 1 and how to match custom skins to the correct in-game categories so they appear in the right place in Create-A-Sim.
