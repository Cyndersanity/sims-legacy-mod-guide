Essential Modding Tools

All free – all from the Sims1Tools GitHub repo unless noted

✅ All Sims1Tools in one place: github.com/trolando/Sims1Tools
BEGINNER Transmogrifier (TMog)
Always start here. Clones any existing game object as a new .iff file so you edit the copy without touching the original. Essential first step for any object mod. Created by Don Hopkins (Maxis). Download from The Sims Depot – the original site is offline.
Sims Depot ↗
INTERMEDIATE IFF Pencil 2
The full .iff editor. View and edit sprites, BHAV behaviour trees, BCON constants, STR# strings, and every other resource inside an .iff file. The main tool for creating complex mods.
Sims1Tools ↗
BEGINNER Constant Contraption
Edits numeric values in .iff files – price, skill gain rate, motive impact. The easiest way to make gameplay tweaks without touching BHAV scripts. Good first mod project.
Sims1Tools ↗
BEGINNER Career Creator 3
Specialised editor for creating custom career tracks. Replaces existing careers – you can't add new ones. Output goes in GameData\Objects\ not Downloads\.
Sims1Tools ↗
BEGINNER Strings Scavenger
Edits text labels – object names, interaction menu text and descriptions. Simpler than editing STR# resources directly in IFF Pencil 2.
Sims1Tools ↗
INTERMEDIATE FARx
Extracts individual .iff files from .FAR container bundles. Use when a mod is packaged as a FAR but specific files need to go in different folders.
Sims1Tools ↗
ADVANCED Simpose / Simposium
Creates custom Sim poses and animation sequences. Advanced – requires understanding the animation system. Tutorials at Awesome Expression.
Awesome Expression ↗
INTERMEDIATE FaithBeam Legacy Hacks
A runtime memory patcher that unlocks edit_char and other hidden debug features. Run it before launching the game each session and keep it open while you play. Works on both Steam and EA App – no Steamless required. Never permanently modifies Sims.exe.
GitHub ↗

CC Creation Tools

Official Maxis tools from the Internet Archive

⚠️ Registry note: These tools look for the game at the old registry path. Legacy Collection creates no registry entries. If a tool can't find your game, see the CC Troubleshooting registry fix.
BEGINNER The Sims Creator
Official Maxis tool for creating custom body skins and head textures. Provides a 3D Sim body to paint onto and exports game-ready .BMP/.CMX/.SKN files. The standard starting point for custom CC.
Archive.org ↗
BEGINNER Home Crafter
Official Maxis tool for custom wall coverings and floor tiles. Import any image and it generates the correct .WLL or .FLR file automatically.
Archive.org ↗
FREE GIMP
Free image editor for skin texture painting. Critical: go Image → Mode → Indexed (256 colours) before exporting as BMP. Regular 24-bit BMPs show as white in-game.
gimp.org ↗
FREE IrfanView
Lightweight free image viewer. Useful for quickly converting BMPs to 8-bit indexed format and batch converting multiple skins at once.
irfanview.com ↗

Community Tools & Resources

PROJECT Simitone
An open-source 3D recreation of The Sims 1 built on the FreeSO engine, using your own Legacy Collection game files as assets. In active development. Not a mod – a separate application.
GitHub ↗
FREE 7-Zip
Essential for Sims 1 modding. Old Maxis "Get Cool Stuff" downloads were distributed as Windows .exe installers - 7-Zip lets you open them as archives and extract files manually without running the installer.
7-zip.org ↗
INTERMEDIATE Steamless
Unpacks the Steam DRM wrapper from Sims.exe to give you a clean unencrypted copy. No longer needed for FaithBeam's debug cheats tool. May be useful for other third-party tools that still require an unencrypted exe.
GitHub ↗

External Tools & Tutorials

Downloads and guides from the community

GitHub – Sims1Tools (trolando)
Essential modding toolkit: IFF Pencil 2, FARx, Career Creator 3, Constant Contraption, Strings Scavenger.
↗ github.com/trolando/Sims1Tools
Skin Doctor Validation
Validates skin files before adding them to the game. Catches incorrect colour depth, malformed files and other issues that cause white blanks in game. Successor to Sims File Cop (which breaks on Hot Date+ saves). Available from BeyondSims and the Internet Archive.
Internet Archive ↗
Sims 1 Tutorials – Awesome Expression
Advanced tutorials covering custom animations, custom food, custom sounds and object cloning with Transmogrifier.
↗ awesomeexpression.com
The Sims Depot
Must-have programs, adding skins and meshes. Good beginner resource for tools and setup.
↗ thesimsdepot.com
Parsimonious – Adding Downloads To Your Game
Step-by-step instructions for installing all different file types for Sims 1 CC, plus troubleshooting tips.
↗ parsimonious.org
