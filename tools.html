Essential Modding Tools

All free – from multiple community sources

✅ Tom van Dijk's modding tools (IFF Pencil 2, FARx, Career Creator 3, Constant Contraption): github.com/trolando/Sims1Tools
✅ Maxis official tools (The Sims Creator, HomeCrafter, Art Studio, FaceLift): Feraligatr installer
BEGINNER Transmogrifier (TMog)
The essential tool for object modding. Clones objects, exports sprite images as editable BMPs, and imports your edited versions back in. Handles the entire retexture workflow. Also lets you edit the object's catalog name, description, and price. Created by Don Hopkins (Maxis).
Sims Depot ↗
INTERMEDIATE IFF Pencil 2
The full .iff editor for behaviour and data resources: BHAV scripts, BCON constants, TTAB interactions, STR# strings, OBJD properties, and more. Does not edit sprites or palettes – use TMog for visual changes. Needs Windows XP SP3 compatibility mode and Run as administrator on Windows 10/11.
Sims1Tools ↗
BEGINNER Constant Contraption
Edits numeric values in .iff files – price, skill gain rate, motive impact. The easiest way to make gameplay tweaks without touching BHAV scripts. Good first mod project.
Sims1Tools ↗
BEGINNER Career Creator 3
Specialised editor for creating custom career tracks. Replaces existing careers – you can't add new ones. Output goes in GameData\Objects\ not Downloads\.
Sims1Tools ↗
BEGINNER Strings Scavenger
Edits text labels – object names, interaction menu text and descriptions. Simpler than editing STR# resources directly in IFF Pencil 2.
Sims1Tools ↗
INTERMEDIATE FARx
Extracts individual .iff files from .FAR container bundles. Use when a mod is packaged as a FAR but specific files need to go in different folders.
Sims1Tools ↗
ADVANCED Simpose / Simposium
Creates custom Sim poses and animation sequences. Advanced – requires understanding the animation system. Tutorials at Awesome Expression.
Awesome Expression ↗
INTERMEDIATE FaithBeam Legacy Hacks
A runtime memory patcher that unlocks edit_char and other hidden debug features. Run it before launching the game each session and keep it open while you play. Works on both Steam and EA App – no Steamless required. Never permanently modifies Sims.exe.
GitHub ↗

CC Creation Tools

Official Maxis tools from the Internet Archive

⚠️ Registry note: These tools look for the game at the old registry path. Legacy Collection creates no registry entries. If a tool can't find your game, see the CC Troubleshooting registry fix.
BEGINNER The Sims Creator
Official Maxis tool for creating custom body skins and head textures. Provides a 3D Sim body to paint onto and exports game-ready .BMP/.CMX/.SKN files. The standard starting point for custom CC.
Archive.org ↗
BEGINNER Home Crafter
Official Maxis tool for custom wall coverings and floor tiles. Import any image and it generates the correct .WLL or .FLR file automatically.
Archive.org ↗
BEGINNER FaceLift Gold
Official Maxis tool for creating new Sim head shapes by morphing and blending existing heads. Shows nine head variants at once; adjust the mutation rate slider to control how drastic the changes are. Use the Fine Tune screen to adjust eyes, nose, and jaw with precision sliders. Export directly into the game. Requires the Feraligatr installer or manual registry fix.
Feraligatr installer ↗
BEGINNER SneakySims Floor Maker
Browser-based floor creator – no download or registry fix needed. Upload a 128×128 image and export a .FLR file directly. Tested with the Legacy Collection.
SneakySims Floor Maker ↗
BEGINNER SneakySims Wallpaper Maker
Browser-based wall creator – companion to the Floor Maker. Upload a 128×232 image (the tool auto-resizes if needed) and export a .WLL file. No download required.
SneakySims Wallpaper Maker ↗
FREE GIMP
Free image editor for skin texture painting. Critical: go Image → Mode → Indexed (256 colours) before exporting as BMP. Regular 24-bit BMPs show as white in-game.
gimp.org ↗
FREE IrfanView
Lightweight free image viewer. Useful for quickly converting BMPs to 8-bit indexed format and batch converting multiple skins at once.
irfanview.com ↗
PAID Photoshop
Adobe's professional image editor. Works for Sims 1 CC if you already have it. To export a valid game BMP: Image → Mode → Indexed Color (256 colours), then File → Save As → BMP. Does not require a registry fix. No free version – GIMP or Paint.NET are free alternatives that do the same job.
FREE Paint.NET
Free image editor for Windows. An alternative to GIMP – simpler interface, fewer features. Supports layers, selections, and BMP export. Does not require a registry fix and works independently of the game. Good for straightforward colour edits if GIMP feels too complex. Note: Paint.NET must also export as Indexed (256-colour) BMP – use the 8-bit option in the Save As dialog.
getpaint.net ↗

3D Modelling & Mesh Tools

For creating custom meshes – clothing, heads, and objects

These tools let you create or edit the actual 3D meshes that skins wrap around, and render object sprites. If you only want to repaint existing shapes, you don't need these – the CC Creation tools above are enough. These are for when you want to change how a Sim's body, hair, or an object actually looks in 3D.

ℹ️ New to mesh editing? Start with Blender + TS1 Blender IO. It's free, actively maintained, and handles all the Sims 1 3D file formats. MilkShape 3D is the classic option if you're following older tutorials.
FREE Blender
Free, open-source 3D modelling software. With the TS1 Blender IO addon installed, it can do everything MilkShape 3D, Skn2Obj, and BodyWarp used to do – in one free tool. Versions 4.1 through 4.5 are supported.
blender.org ↗
RECOMMENDED TS1 Blender IO (by mixiate)
Blender addon that imports and exports all Sims 1 3D file formats: SKN meshes, BCF skeletons, CMX configs, and CFP animations. Auto-loads textures from your game directory. Supports animation events and CFP compression. The modern replacement for MilkShape + Skn2Obj.
GitHub ↗
INTERMEDIATE TS1 Renderer (by mixiate)
Blender addon for rendering object sprites. Automatically renders all rotation angles and zoom levels, generates z-buffer depth sprites, and handles multi-tile objects. Requires TS1 Compiler for final assembly into game-ready IFF files. Blender 4.1, 4.3–4.5 supported (not 4.2 due to a Blender bug).
GitHub ↗
INTERMEDIATE TS1 Compiler (by mixiate)
Companion to TS1 Renderer. Takes rendered sprite images and assembles them into complete, game-ready IFF object files. Handles sprite indexing, z-buffers, and IFF packaging for you.
GitHub ↗
CLASSIC MilkShape 3D
The standard Sims 1 mesh editor from the original modding era (2000–2010). Has native SKN import/export, skeleton/bone viewing, and weight painting. $25 shareware with 30-day trial. Most older tutorials assume this tool. Still works but no longer updated.
milkshape3d.com ↗
CLASSIC Skn2Obj
Converts SKN mesh files to OBJ format and back. Needed if you want to edit Sims 1 meshes in a 3D editor that doesn't have native SKN support. Not needed if using TS1 Blender IO or MilkShape (both handle SKN natively). Available from BeyondSims archives.
CLASSIC BMF2SKN
Converts binary mesh files (.BMF format, found packed inside older FAR archives) into the standard SKN format that can be edited normally. Only needed when working with older Sims 1 content that contains BMF files. Created by Aladrin Kelahn. Available from BeyondSims archives.
CLASSIC BodyWarp
Quick body part manipulation – rotate, scale, and move individual body segments without a full 3D editor. Can also generate CMX configuration files. Useful for simple adjustments like lengthening legs or resizing heads.
FREE UVMapper Classic
Remaps texture coordinates on meshes. Useful when creating textures for custom meshes – lets you see exactly how a flat BMP wraps around the 3D shape. Free classic edition available.
uvmapper.com ↗
FREE SimShow
Standalone Sims 1 mesh viewer. Load a SKN file with a BMP texture to preview how the texture wraps onto the 3D mesh before putting it in game. Essential for checking new or edited meshes for holes, misaligned textures, and other problems without launching the full game. Available from BeyondSims archives.

Community Tools & Resources

PROJECT Simitone
An open-source 3D recreation of The Sims 1 built on the FreeSO engine, using your own Legacy Collection game files as assets. In active development. Not a mod – a separate application.
GitHub ↗
FREE EliSims
Neighbourhood manager – switch between multiple neighbourhoods, preview skins, find missing skin files, import families, and manage floors/walls/objects. Lightweight Windows utility. Has not been fully tested with the Legacy Collection but may be useful for skin management.
EliSims ↗
FREE 7-Zip
Essential for Sims 1 modding. Old Maxis "Get Cool Stuff" downloads were distributed as Windows .exe installers - 7-Zip lets you open them as archives and extract files manually without running the installer.
7-zip.org ↗
INTERMEDIATE Steamless
Unpacks the Steam DRM wrapper from Sims.exe to give you a clean unencrypted copy. No longer needed for FaithBeam's debug cheats tool. May be useful for other third-party tools that still require an unencrypted exe.
GitHub ↗

External Tools & Tutorials

Downloads and guides from the community

GitHub – Sims1Tools (trolando)
Essential modding toolkit: IFF Pencil 2, FARx, Career Creator 3, Constant Contraption, Strings Scavenger.
↗ github.com/trolando/Sims1Tools
Skin Doctor Validation
Validates skin files before adding them to the game. Catches incorrect colour depth, malformed files and other issues that cause white blanks in game. Successor to Sims File Cop (which breaks on Hot Date+ saves). Available from BeyondSims and the Internet Archive.
Internet Archive ↗
Sims 1 Tutorials – Awesome Expression
Advanced tutorials covering custom animations, custom food, custom sounds and object cloning with Transmogrifier.
↗ awesomeexpression.com
The Sims Depot
Must-have programs, adding skins and meshes. Good beginner resource for tools and setup.
↗ thesimsdepot.com
Parsimonious – Adding Downloads To Your Game
Step-by-step instructions for installing all different file types for Sims 1 CC, plus troubleshooting tips.
↗ parsimonious.org
