What Is the Legacy Collection?

The Sims Legacy Collection is the 2025 re-release of the original The Sims (2000) and all seven expansion packs, in a single digital purchase. Released January 31, 2025 on Steam, EA App, and Epic Games Store. It is digital only - no physical or boxed edition was released. It replaces the older Complete Collection as the standard way to buy the original game new.

INCLUDED What's included
Base game (2000) plus all 7 expansion packs: Livin' Large, House Party, Hot Date, Vacation, Unleashed, Superstar, and Makin' Magic. Also grants The Sims 4 with the Throwback Fit Pack on the same EA account. Digital only - available on Steam, EA App and Epic Games Store. No physical or boxed edition was released.
NOT INCLUDED Licensed content removed
Branded objects (McDonald's cart, Pepsi machine, Intel computer) and licensed Simlish songs were removed when their licenses expired. These are fully archived and can be added manually - see the Missing Maxis Content section below.
ℹ️ The Legacy Collection runs on the same underlying game code as the original. Almost all mods and CC made for the original Sims 1 still work, with path adjustments and a few technical differences covered on this page.

Technical Changes vs the Original

What actually changed under the hood

🖥️
New renderer: Vulkan replaces DirectDraw
The biggest change. The original game used DirectDraw and DirectX 6, which caused severe compatibility issues on modern Windows. Legacy Collection replaces this with Vulkan, giving full GPU acceleration and 24-bit colour. This is why Legacy requires a Vulkan-compatible GPU - most GPUs from around 2012 onwards qualify. If you get a "Vulkan Initialization Failed" error on launch, see the Game Troubleshooting page.
💾
Save files moved to a dedicated folder
In the original game, UserData folders sat inside the game's install directory. Legacy Collection moves them to C:\Users\%USERPROFILE%\Saved Games\Electronic Arts\The Sims 25\. On first launch the game copies the default neighbourhood templates there. Steam Cloud saves are not supported - back up this folder manually if you play on multiple machines.
📐
Resolution scaling overhauled (March 2025 patch)
The game now renders onto an internal canvas at the next clean integer scale up from your chosen resolution. The window is freely resizable, but the effective in-game resolution only updates on restart. For the sharpest image, choose a resolution that divides cleanly into your monitor's native resolution.
📋
No Windows Registry entries
The original game wrote its install path to the Windows Registry. Older CC creation tools (The Sims Creator, Home Crafter) use this to find the game automatically. Legacy Collection creates no registry entries. Old tools may fail to find your game or save output to the wrong location. See CC Troubleshooting for the registry fix.
☁️
No Steam Cloud saves
Save files are not synced via Steam Cloud. Back up C:\Users\%USERPROFILE%\Saved Games\Electronic Arts\The Sims 25\ manually before reinstalling or switching machines. Some mods write data to save files permanently, so keeping a backup is especially important.
🔒
Steam executable is DRM-protected
The Steam version's Sims.exe is wrapped in Steam DRM. Mods that directly patched the original exe will not work. FaithBeam's Legacy Collection Hacks tool unlocks the edit_char debug cheat by patching the game in memory at runtime - it does not permanently modify any files. Run Sims1LegacyHacks.exe before launching the game each session and keep it open while you play. Works on both Steam and EA App versions - no Steamless required. See the Must-Have Mods page.
🖱️
Native mouse wheel zoom
The original Sims 1 had no native scroll wheel support — you zoomed using keyboard shortcuts or on-screen buttons. Legacy Collection adds native mouse wheel zooming. Scroll up to zoom in, scroll down to zoom out, across all view modes.
⌨️
Alt+Enter cycles resolution scale modes
Pressing Alt+Enter while the game is running cycles through the available display modes for your monitor. On 1080p displays: fullscreen and windowed. On 1440p: 1x and 2x scale. On 4K: 1x, 2x and 3x scale. The number of modes available depends entirely on your monitor resolution.
🔐
Configuration files are encrypted
The original Complete Collection stored settings in editable plain text .ini files. The Legacy Collection encrypts its config files — you cannot manually edit settings outside what the in-game options menu (F5) provides. This also means some power-user tweaks possible in the original are not available in Legacy.
🎬
EA intro video removed
The original game opened with an EA/Maxis intro video sequence. Legacy Collection skips this entirely and goes straight to the main menu. The -skip_intro launch flag has no effect since there is no intro to skip. A Reddit workaround exists to manually restore the intro video for the Complete Collection version, but this does not work with Legacy Collection — EA changed something in the exe that prevents it.

RAM4 GBThe original game used very little memory by modern standards
Storage~5 GBPlus mods and CC - keep Downloads folder under 2.5 GB
PlatformSteam, EA App, or Epic GamesDigital only, no disc version
Steam DeckSupportedCommunity reports consistently positive - game runs well on Deck
⚠️ Older Intel integrated graphics warning: Intel HD Graphics chips older than Iris Xe (pre-2020, typically 4th-8th gen Core) often do not support Vulkan. If you have only an older Intel integrated GPU, the Legacy Collection may not run. The original Complete Collection (DirectDraw) will work on these systems, but Legacy Collection requires Vulkan.

Missing Official Maxis Content

The "Get Cool Stuff" downloads not included in Legacy

Between 2000 and 2003, Maxis ran a "Get Cool Stuff" section on the official Sims website with free downloadable objects, skins, tools and houses. Most of this was not included in Legacy Collection - either because licenses expired (branded objects) or it was simply not bundled. The community has archived all of it and made it Legacy-compatible.

⚠️ Do not run the original .exe installers. The original Maxis downloads were distributed as Windows installers that look for the game in the old Complete Collection registry path. They will fail silently on Legacy. Use the GitHub archive below instead - files are pre-extracted and ready to copy.
RECOMMENDED Legacy-ready archive
alon-abadi on GitHub has packaged all known Maxis "Get Cool Stuff" content specifically for Legacy Collection. Files are extracted and organised by destination folder. Includes objects (McDonald's cart, Pepsi machine, Intel computer etc.), promotional skins, NPC templates and official houses.

ARCHIVE Full original collection
Complete archive of the original Maxis site content on Internet Archive - all objects, tools, skins, homes and graphics packs. Contains original .exe files. Open them with 7-Zip as archives rather than running them - extract and copy files manually.

ℹ️ How to install from the GitHub archive: Download the ZIP, extract it, and copy files to the folders shown in the folder names. Objects to Downloads\, skins to GameData\Skins\ etc. No installer needed.
